# 📄 Currículo Fictício Web

Este projeto consiste na criação de um currículo fictício desenvolvido como página web, utilizando HTML e CSS.

O objetivo foi praticar a estruturação de páginas, estilização e organização de informações de forma clara e profissional.

---

## 🚀 Objetivo

Desenvolver um currículo online com foco em:

- Estrutura semântica em HTML  
- Estilização com CSS  
- Organização visual de informações  
- Boas práticas de layout  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 📌 Funcionalidades

- Exibição de informações pessoais  
- Seção de habilidades  
- Experiências profissionais (fictícias)  
- Formação acadêmica  
- Layout organizado e responsivo (se tiver, pode manter)
